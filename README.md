# Skillet Landing Page

Адаптивный лендинг, сверстанный по макету и подготовленный как демонстрационный проект для портфолио.  
Проект выполнен в чистом HTML, CSS и JavaScript.

## 🚀 Демо

Готовая версия сайта доступна по ссылке:  
https://our-place-practice.vercel.app/

## 🛠 Технологии

- HTML5
- CSS3 (адаптивная верстка, BEM)
- JavaScript
- Flexbox / Grid
- Кроссбраузерная поддержка

## 📱 Адаптивность

Сайт корректно отображается на:

- мобильных устройствах
- планшетах
- ноутбуках
- широких мониторах

## ✨ Что реализовано

- Полная адаптивная верстка всех секций
- Анимации и интерактивные элементы
- Чистая файловая структура
- Семантическая разметка
